# Hướng dẫn chạy code
1. Mở 2 terminal chạy backend và frontend. Lệnh: cd frontend, cd backend
2. npm install từng thư mục
3. Trở về thư mục gốc: cd ..

# Chạy chatbot
1. cd ollama
2. pip install -r requirements.txt
3. streamlit run app.py
